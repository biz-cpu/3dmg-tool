# SC 3Dマシンガイダンス 刃先精度確認アプリ

## 概要
SC 3Dマシンガイダンスの刃先精度不良時に行う事前確認チェック・刃先座標記録をデジタル化したStreamlitアプリです。

## 機能
| ページ | 内容 |
|--------|------|
| **事前確認チェック** | 6カテゴリ・全50項目以上のOK/NGチェック、数値入力（衛星数・遅延時間等） |
| **刃先座標チェック** | GNSS精度情報、基準点情報、4姿勢の座標差分入力、±5cm自動判定、傾向分析 |
| **サマリー・レポート** | 全項目の集計、JSONダウンロード |

## セットアップ・起動

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ファイル構成
```
app.py            # メインアプリ（単一ファイル）
requirements.txt  # 依存ライブラリ
README.md         # このファイル
```

## 注意事項
- ログ取得関連（⑤シート）は本アプリの対象外です（別途対応）
- セッションデータはブラウザを閉じると消えます。JSONダウンロードで保存してください。
- チェックボックス判定基準：OK = 問題なし確認済み / NG = 問題あり要対応

## ±5cm 判定基準
```
|差分| ≦ 0.05m → OK（緑）
|差分| ＞ 0.05m → NG（赤）
```
